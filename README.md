# Sweet Treats Bakery - Full Stack Web Application

A custom-built web platform for "Sweet Treats" bakery. This project allows the bakery owner to manage a dynamic daily menu and collect customer feedback, while providing a clean and intuitive interface for visitors.

## Key Features
- **Dynamic Daily Menu**: Display of pastries and cakes with names, descriptions, and prices.
- **Admin Dashboard**: Secure login area for the owner to update the menu and manage feedback.
- **Customer Feedback System**: A form for users to leave reviews. 
- **Privacy Protection**: Customer emails are stored in the database but hidden from the public feedback page.
- **Latest Updates**: The homepage automatically displays the 3 most recent customer reviews.
- **Search & Filtering**: Users can search for specific menu items by name.

## Tech Stack
- **Backend**: PHP (Server-side logic & Form handling)
- **Database**: MySQL (Storage for menu items and customer feedback)
- **Frontend**: HTML5, CSS3 (Clean and responsive layout)
- **Security**: Admin authentication and data sanitization.
